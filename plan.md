# Plan: Chicago AI-Local Web Factory (Control Tower)

## Mission Statement
시카고 로컬 소상공인의 실시간 작업 로그(Plan, Context, Todo)를 자동화된 고성능 AI 콘텐츠 마케팅 엔진으로 전환하여, 최소의 관리 인풋으로 최대의 온라인 매출 가치와 브랜드 가시성을 창출한다.

---

## 🎯 [약문통 원해자] 전략 프레임워크

### 1. [약] 약속 (Promise)
"3일 만에 완성되는 고성능 AI 결합 웹사이트와 실시간 자동 블로그 마케팅 시스템을 통해, 코딩이나 글쓰기 한 줄 없이도 시카고 로컬 시장에서의 온라인 가시성과 고객 유치 효과를 극대화해 드립니다."

### 2. [문] 문제 (Problem)
시카고의 로컬 비즈니스(예: 웨딩 사진작가 Olivia Choi)들은 대형 웹 에이전시의 과도한 비용($5,000 이상)과 느린 제작 속도(최소 4~8주)에 큰 비용 부담을 겪고 있습니다. 게다가 웹사이트를 어렵게 개설한 이후에도 본업에 바빠 포트폴리오 업데이트나 블로그 마케팅, SNS 홍보용 콘텐츠를 작성할 시간과 노하우가 전무합니다.

### 3. [통] 통념 (Conventional Wisdom)
많은 소상공인들이 "템플릿 기반 빌더(Wix, WordPress 등)로 사이트를 대충 꾸며놓기만 하면 알아서 구글에서 검색되고 손님이 찾아올 것"이라고 믿거나, "직접 마케팅 글을 써서 뿌려야 한다"고 생각합니다.

### 4. [원] 원인 (Cause)
실제 원인은 두 가지입니다.
1. 템플릿 빌더들은 무거운 JavaScript 로딩으로 인해 구글 Core Web Vitals(웹 속도 성능) 점수가 낮아 SEO(검색 최적화) 랭킹에서 밀려납니다.
2. 로컬 비즈니스의 진짜 경쟁력은 '현장 스토리(현지 웨딩 촬영기, 현장 시공기 등)'에 있으나, 소상공인이 본업을 수행하며 이를 마케팅에 적합한 스토리로 변환하고 SNS 채널로 유통하는 과정이 가혹할 정도로 번거롭기 때문에 중도 포기하게 됩니다.

### 5. [해] 해법 (Solution)
Astro.build 기반의 Zero-JS 초고속 웹 아키텍처와 Firebase Hosting의 초고속 CDN을 탑재한 사이트를 3일 만에 팩토리 형태로 빌드합니다. 그리고 작업자가 제공한 3대 핵심 파일(`plan.md`, `context.md`, `todo.md`)만으로 AI(Gemini API)가 매력적인 클라이언트용 스토리텔링 블로그 포스트 및 SNS(LinkedIn, Instagram) 배포용 요약본을 즉각 생산하고 최적의 SEO 마크업을 입히는 **"Chicago AI-Local Web Factory"**를 구축합니다.

### 6. [자] 자각 및 미션 (Self-Awareness & Mission)
"내가 해결해야 했던 문제는 단순히 웹사이트라는 정적 껍데기를 만드는 것이 아니라, 매번 현장에서 흘리는 나의 땀방울(작업 로그)을 돈이 되는 스토리와 유기적인 마케팅 채널로 자동 전환하는 엔진을 구축하는 것이었구나!"
- **Mission Statement**: **"시카고 소상공인의 생생한 비즈니스 스토리를 AI 자동화 엔진으로 연결해 영속적인 온라인 성장을 실현한다."**
